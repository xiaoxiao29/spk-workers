## 快速开始

### 前提条件

- 一个 Cloudflare 账号
- Node.js 18+ 环境

### 步骤概览

```
1. 配置 Cloudflare 资源（R2、KV）
2. 克隆并配置项目
3. 部署 Workers
4. 上传设备配置文件
5. 在 Synology 上添加仓库
```

## 配置 Cloudflare 资源

### 1. 创建 R2 Bucket

1. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)
2. 进入 **R2** 服务
3. 创建 Bucket，名称设为 `spks`

### 2. 创建 KV Namespace

1. 进入 **Workers & Pages**
2. 创建 **KV Namespace**，名称设为 `SPKS_CACHE`

### 3. 绑定资源到 Workers

资源绑定通过 `wrangler.toml` 配置完成，无需手动在 Dashboard 操作。

在 `wrangler.toml` 中配置：

```toml
[[r2_buckets]]
binding = "SPKS_BUCKET"
bucket_name = "spks"

[[kv_namespaces]]
binding = "SPKS_CACHE"
id = "your-kv-namespace-id"  # 从 KV Namespace 页面获取
```

部署时 wrangler 会自动绑定资源。

## 部署 Workers

### 1. 克隆项目

```bash
git clone https://github.com/your-repo/spk-workers.git
cd spk-workers
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置 wrangler.toml

复制配置文件模板：

```bash
cp examples/wrangler.toml.example wrangler.toml
```

然后编辑 `wrangler.toml`，填入您的实际配置值：

- `kv_namespaces[].id` - KV Namespace ID
- `d1_databases[].database_id` - D1 数据库 ID

### 4. 设置 API Key

API Key 是您自定义生成的密钥，用于保护上传接口。

**生成密钥：**

```bash
# 方式 1：openssl（推荐）
openssl rand -base64 32

# 方式 2：Node.js
node -e "console.log(require('crypto').randomBytes(32).toString('base64'))"
```

**设置到 Workers：**

```bash
wrangler secret put SSPKS_API_KEY
# 输入您生成的密钥
```

> ⚠️ 生产环境请使用强密钥，不要使用简单密码

### 5. 部署

```bash
npm run deploy
```

## 配置设备列表

部署时会自动上传设备配置文件到 R2。

如需单独上传：

```bash
npm run upload:config
```

或手动上传：

```bash
wrangler r2 object put spks/conf/synology_models.yaml --file=conf/synology_models.yaml
```

## 在 Synology 上添加仓库

### 步骤 1：打开 Synology Package Center

登录您的 Synology NAS，打开 **Package Center**。

### 步骤 2：添加自定义仓库

1. 点击 **设置** → **程序包来源**
2. 点击 **新增**
3. 填写以下信息：

| 字段 | 值 |
|------|-----|
| 名称 | `SPK Workers` |
| URL | `https://your-worker.workers.dev` |

> ⚠️ 将 `your-worker.workers.dev` 替换为您的实际地址

### 步骤 3：验证连接

点击确定后，Package Center 会自动获取包列表。

## 上传 SPK 包

### 通过 Web 界面上传

1. 访问 `/upload` 页面
2. 输入您的 **API Key**
3. 拖拽或选择 SPK 文件
4. 点击上传

### 通过命令行上传

```bash
curl -X POST https://your-worker.workers.dev/api/upload \
  -H "X-API-Key: your-api-key" \
  -F "spk=@/path/to/your-package.spk"
```

| 缓存服务 | Cloudflare KV |
