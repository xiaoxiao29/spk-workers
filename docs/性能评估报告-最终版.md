## 4. 缓存策略评估

### 4.1 多级缓存架构

**缓存层级**：

1. **L1: 全局变量缓存（Worker 内存）**
   - cachedDeviceList
   - routerInstance

2. **L2: KV 缓存（边缘节点）**
   - 设备配置缓存
   - 包元数据缓存
   - 静态资源缓存

3. **L3: HTTP 缓存（浏览器/CDN）**
